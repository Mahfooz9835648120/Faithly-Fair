# Faithly Fair

Responsive React + TypeScript gift storefront with Supabase catalog/content, transactional guest checkout, COD and exact-amount UPI flows, private payment proofs, order emails, WhatsApp support, and an RLS-protected admin studio.

```bash
npm install
npm run dev
```

The app has a polished offline demo state. Follow [SETUP.md](./SETUP.md) to enable live orders and admin access.
